# Power BI 入门

开始使用 Power BI Desktop 创建你的第一个报告。

## 安装 Power BI Desktop

1. 访问 [Power BI 官网](https://powerbi.microsoft.com/)
2. 下载 Power BI Desktop
3. 安装并启动应用程序

## 界面概览

### 主要视图

**报表视图 (Report View)**
- 创建可视化和报告的主要工作区
- 拖放字段创建图表

**数据视图 (Data View)**
- 查看和编辑数据表
- 创建计算列

**模型视图 (Model View)**
- 管理表之间的关系
- 查看数据模型结构

## 第一个报告：销售分析

### 步骤 1：获取数据

```
1. 点击 "获取数据" (Get Data)
2. 选择数据源类型（例如：Excel）
3. 浏览并选择文件
4. 在导航器中选择要导入的表
5. 点击 "加载" 或 "转换数据"
```

### 步骤 2：转换数据（Power Query）

在 Power Query 编辑器中清洗数据：

```
- 删除不需要的列
- 更改列类型
- 删除空行
- 替换值
- 添加自定义列
```

示例：添加计算列

```
// 在 Power Query 中添加自定义列
Revenue = [Quantity] * [Unit Price]
```

### 步骤 3：创建可视化

#### 创建柱状图

1. 选择 **柱状图** 视觉对象
2. 将 **Category** 拖到轴
3. 将 **Sales** 拖到值

#### 创建折线图

1. 选择 **折线图** 视觉对象
2. 将 **Date** 拖到轴
3. 将 **Revenue** 拖到值

#### 创建卡片

1. 选择 **卡片** 视觉对象
2. 将 **Total Sales** 度量值拖到字段

### 步骤 4：添加筛选器和切片器

#### 添加切片器

```
1. 选择 "切片器" 视觉对象
2. 添加字段（如 Date、Category）
3. 调整切片器样式
```

#### 使用筛选器

```
- 视觉对象级别筛选器
- 页面级别筛选器
- 报告级别筛选器
```

### 步骤 5：格式化报告

#### 更改主题

```
1. 视图 > 主题
2. 选择预定义主题或自定义
```

#### 调整视觉对象格式

```
- 标题和标签
- 数据颜色
- 背景和边框
- 工具提示
```

## 常用视觉对象

### 基础图表

| 类型 | 用途 | 示例 |
|------|------|------|
| 柱状图 | 比较类别数据 | 各产品销售额 |
| 折线图 | 显示趋势 | 月度销售趋势 |
| 饼图 | 显示占比 | 市场份额 |
| 表格 | 显示详细数据 | 交易明细 |
| 矩阵 | 交叉表分析 | 区域×产品销售 |

### 高级视觉对象

- **地图**：地理数据可视化
- **仪表盘**：KPI 监控
- **瀑布图**：累计变化分析
- **散点图**：相关性分析
- **树状图**：层次结构数据

## 数据刷新

### 手动刷新

```
点击 "刷新" 按钮更新数据
```

### 自动刷新（需要 Power BI Pro）

```
1. 发布报告到 Power BI Service
2. 配置数据源凭据
3. 设置刷新计划
```

## 发布报告

### 发布到 Power BI Service

```
1. 点击 "发布"
2. 登录 Power BI 账户
3. 选择工作区
4. 等待发布完成
```

### 创建仪表板

```
1. 在 Power BI Service 中打开报告
2. 将视觉对象固定到仪表板
3. 创建新仪表板或选择现有仪表板
```

## 最佳实践

- ✅ 在 Power Query 中完成数据清洗

- ✅ 删除不需要的列以提高性能

### 数据建模
- ✅ 使用星型架构
- ✅ 创建日期表
- ✅ 建立正确的表关系

### 可视化设计
- ✅ 保持简洁明了
- ✅ 使用一致的颜色主题
- ✅ 添加清晰的标题和标签
- ✅ 避免图表过度拥挤
